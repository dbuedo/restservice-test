# REST Service test 
Example project of very basic REST Service developed with:
 * Jersey 2.16
 * Spring Framework 4.1
 * Hibernate 4.3
 * HSQLDB 2.3
 
## REST Resources deployed
Examples of requests:
 * GET /status
   * To check if the service is up.
 * GET /call
   * Returns a list of Calls
 * GET /call/1
   * Returns a Call
